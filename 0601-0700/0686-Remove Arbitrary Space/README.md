# 686. Remove Arbitrary Space

Difficulty: Easy

https://www.lintcode.com/problem/remove-arbitrary-space/description

**Description**  
Remove arbitrary spaces from a sentence

**Example 1:**
```
Input: s = "The  sky   is blue"
Output: "The sky is blue"
```

**Example 2:**
```
Input: s = "  low               ercase  "
Output: "low ercase"
```
